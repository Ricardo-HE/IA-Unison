\section{Tipos mutables}

\begin{enumerate}
\item Mira el siguiente código y ejecutalo:
\begin{verbatim}
# Vamos a generar 3 listas que parecerían iguales
lista_1 = [1, 2, 3, "toto", ["a", "b", "c"]]
lista_2 = lista_1
lista_3 = lista_1[:]

lista_2[0] = "XXX"
print(lista_2)

lista_3[-1][0] = 1000
print(lista_3)
\end{verbatim}

  Ahora responde (sin revisar) lo que crees que debería salir con
\begin{verbatim}
print(lista_1)
print(lista_2)

# Ejemplo:
# print(lista_3)  -->  [1, 2, 3, 'toto', [1000, 'b', 'c']]
\end{verbatim}
  Una vez que lo contestaste, ejecuta las operaciones y explica el
  porqué de los resultados.

\item Revisa el siguiente código
\begin{verbatim}
class coordenadas:
    def __init__(self, x=0, y=0):
        "Inicializa un objet coordenadas"
        self.x = x
        self.y = y

    def __str__(self):
        return "\n\tCoordenada x = {}\n\tCoordenada y = {}".format(self.x, self.y)


a = coordenadas(3, 4)
b = a
b.x = 100
print("b = " + str(b))
\end{verbatim}
  Y ahora escribe que crees que debería salir (sin revisar) con
\begin{verbatim}
print("a = " + str(a))
\end{verbatim}
Una vez que lo contestaste, revisa el resultado y explícalo brevemente.
  
\end{enumerate}

\section{Listas y diccionarios}

\begin{enumerate}
\item Escribe, en una sola linea, una expresión que genere todos los
  números enteros que se encuentran entre $1$ y $1000$ que sean
  divisibles por $2$, $3$, $5$ y $7$ al mismo tiempo. Aprovecha de las
  ventajas para crear listas de:
\begin{verbatim}
	[expresión  for elemento in lista],
	[expresión  for elemento in lista if condición]
\end{verbatim}
 
\item Escribe una función que reciba una lista de elementos (letras,
  numeros, lo que sea), cuente la ocurrencia de cada elemento en la
  lista y la devuelva en forma de diccionario e imprima un histograma
  de ocurrencias, por ejemplo:
\begin{verbatim}
>>> d = FuncionEjemplo( [1,'a',1, 13, 'hola', 'a', 1, 1, 'a', 1], Imprime = True)

1               *****   (5 -> 50%)
'a'             ***     (3 -> 30%)
13              *       (1 -> 10%)
'hola'          *       (1 -> 10%)

>>> print d
{1:5, 'a':3, 13:1, 'hola':1}		
\end{verbatim}

\item Escribe una función que modifique un diccionario y regrese el
  diccionario modificado y una copia del original, donde cada entrada
  del diccionario sea una lista de valores, ejemplo de la función:
\begin{verbatim}
>>> dic1 = {'Pepe':[12, 'enero', 1980], 'Carolina':[15,'mayo',1975],'Paco':[10,'nov',1970]}
>>> dic2 = fundicos(dic1, 'Pepe', 1, 'febrero')
>>> print dic1 
{'Pepe':[12, 'enero', 1980], 'Carolina':[15,'mayo',1975],'Paco':[10,'nov',1970]}
>>> print dic2 
{'Pepe':[12, 'febrero', 1980], 'Carolina':[15,'mayo',1975],'Paco':[10,'nov',1970]}
\end{verbatim}

\end{enumerate}

\section{Funciones y clases}

\begin{enumerate}

\item Escribe una función \texttt{ fun1 } que reciba un número $n$ y
  calcule el número primo inmediatamente superior. Escribe una función
  \texttt{ fun2 } que reciba como argumento un numero y una función, y
  devuelva una lista con la evaluación de la función desde $1$ hasta
  $n$. Prueba la función con \texttt{fun1} y con \texttt{math.sqrt}.

\item Escribe una función, lo más compacta posible, que escoja entre
  los 3 patrones ascii a continuación, e imprima en pantalla el
  deseado, pero de la dimensión $n$ deseada ($n \ge 4$, toma en cuanta
  que para algunos valores de $n$ habrá algún(os) patrones que no se
  puedan hacer).
\begin{verbatim}
          *             ++++           oooooooo
          **            ++++           ooo  ooo
          ***           ++++           oo    oo
          ****          ++++           o      o
          *****             ++++       o      o
          ******            ++++       oo    oo
          *******           ++++       ooo  ooo
          ********          ++++       oooooooo
\end{verbatim}


\item Diseña una clase Matriz e implementa con sobrecarga la suma de

\end{verbatim}
\end{enumerate}
